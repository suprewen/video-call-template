# 视频通话模板生成器

一个在浏览器中运行的视频编辑器：保留原视频，叠加视频通话风格界面，并在右上角保留纯绿区域以便后期合成。

## 功能

- 在网页直接上传视频；
- 自动裁切为 9:16 视频通话画面；
- 保留原视频的画面与声音；
- 叠加视频通话风格控件；
- 在右上角保留 `#00FF00` 纯绿区域；
- 可在“高级选项”调整绿幕和通话 UI 的大小；
- 在浏览器本地合成并下载 MP4；
- 视频不会上传至服务器。

## 浏览器导出

首次导出会加载约 31MB 的浏览器渲染引擎。视频始终在当前设备上处理。

建议使用桌面版 Chrome 或 Safari。较长视频、4K 视频和低性能设备需要更长处理时间。

## 本地开发

```bash
npm install --include=dev
npm run dev
```

## 本地渲染（备用）

浏览器端导出不可用时，也可在本地使用 Remotion：

```bash
npm run render -- /绝对或相对路径/人物视频.mp4
```

可指定输出文件：

```bash
npm run render -- ./input.mp4 ./out/video-call.mp4
```

输出为 720×1280、30fps、H.264 MP4。渲染前会先将输入文件标准化为浏览器可渲染的 H.264，再交由 Remotion 合成；临时文件位于 `public/render-input.mp4`，已被 Git 忽略。
