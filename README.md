# 直播点单视频模板

一个运行在浏览器里的直播点单视频编辑器：**保留原视频**，叠加视频通话风格 UI，并在右上角预留纯绿商品位，供 OBS / 直播伴侣色度键叠加商品、菜单或订单页面。

## MVP 能做什么

- 直接在网页上传人物视频；
- 自动裁切为 9:16 视频通话画面；
- 原样保留人物、动作、口型与声音；
- 固定底部通话控制 UI；
- 右上角固定 `#00FF00` 绿幕商品位；
- 在浏览器本地合成并直接下载 MP4；
- 视频不会上传到 Cloudflare 或其他服务器。

## 浏览器导出

首次点击“导出 MP4”时，网页会下载约 31MB 的 FFmpeg WebAssembly 引擎。之后视频在当前浏览器本地合成。

建议使用桌面端 Chrome 或 Safari。较长视频、4K 视频和低性能设备会需要更长时间；MVP 的目标素材是竖屏、时长较短的人物视频。

## 本地开发

```bash
npm install --include=dev
npm run dev
```

## 本地 Remotion 渲染（备用）

浏览器端导出不可用时，也可以在本地使用 Remotion：

```bash
npm run render -- /绝对或相对路径/人物视频.mp4
```

可指定输出文件和通话名称：

```bash
npm run render -- ./input.mp4 ./out/order-call.mp4 "老张"
```

输出为 720×1280、30fps、H.264 MP4。渲染前会先将输入文件标准化为浏览器可渲染的 H.264，再交由 Remotion 合成；临时文件位于 `public/render-input.mp4`，已被 Git 忽略。

## 直播叠加商品

1. 把生成的 MP4 添加进 OBS 或直播伴侣。
2. 对该视频图层添加“色度键 / Chroma Key”。
3. 选取绿色 `#00FF00`。
4. 将商品图、菜单或点单网页图层置于视频图层下方。

## 部署说明

Cloudflare Pages 托管整个编辑器。视频导出不使用 Pages 服务端算力，完全由用户浏览器在本地完成。
